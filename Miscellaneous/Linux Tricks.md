# Connect to Wifi from Terminal
https://www.linuxfordevices.com/tutorials/ubuntu/connect-wifi-terminal-command-line
1. `ip a` or `iwconfig`
2. `sudo ifconfig wlan0 up`
3. `nmcli dev status`
4. `nmcli dev wifi list`
5. ``sudo` `nmcli --ask dev wifi connect <SSID>``
